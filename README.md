

Ce projet a été développé sous React JS  

pour installer react sur un ordinateur il faut dabord installer 
 	 - node.js  (https://nodejs.org/fr/download/) 
 	 puis
 	 - React JS  (https://react-cn.github.io/react/downloads.html)

pour démarrer l'application il faut ouvrir un invite de commande puis aller dans le dossier ou se trouve le projet 
exécuter les commandes suivantes : 
 	 - npm install 
 	 - npm start 
 votre navigateur par défaut est démarrer et vous pouvez voir l'application 

sur la page d'accueil vous avez la liste des employés de l'entreprise, récupérer grâce a un api sur ce le lien https://reqres.in  
en outre vous avez les options suivantes : 
	- ajouter un nouvel employé en haut à droite 
	- faire les mises à jour sur l'employé sélectionné : 
		*  voir les détails, toutes les informations sur l'employé 
		*  modifier les informations de l'employé 
		*  supprimé l'employé 

les données sont regroupés par page pour permettre de défiler facilement dans la liste
